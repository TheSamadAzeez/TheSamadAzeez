## 🧑‍💻 About Me

I'm a software engineer with **5 years** deep in the **TypeScript & React ecosystem** — long enough to have made peace with the fact that half the job is naming things and the other half is arguing about state management.

I build across the whole stack: **web** with Next.js or TanStack Router, **mobile** with React Native / Expo, and **backend** with Node.js & NestJS. My thing is shipping stuff that survives contact with real users — clean where it counts, pragmatic everywhere else.
